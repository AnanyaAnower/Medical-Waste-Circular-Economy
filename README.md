MedCycle AI: Intelligent Ledger for Medical Waste Resource RecoveryThe ProblemAs highlighted in the New Clean Dhaka Master Plan 2018-2032, Dhaka is facing a critical landfill shortage. Medical waste management costs are rising, and hospitals currently pay high fees for hazardous waste disposal. There is a significant gap in "intermediate treatment" and source segregation.The SolutionMedCycle AI is a circular economy framework that transforms hospital waste from a financial liability into a resource. By using Computer Vision (AI) and a Real-Time Firebase Ledger, the system identifies recyclable medical polymers at the point of disposal and calculates their economic value.Key Features1. AI-Driven Material ValuationThe system uses a two-gatekeeper model to identify recyclables (Saline bags, IV tubes, Plastics). Unlike static systems, this project introduces a Quality Grading Scale (A-E)—inspired by technical assessment methodologies—to price waste based on its purity and condition.2. Automated ROI EngineThe dashboard calculates a real-time Return on Investment based on:Direct Revenue: Market value of recovered high-grade medical polymers.Avoided Costs: Savings generated by diverting waste from expensive hazardous incineration streams (using a 0.8 Cost-Avoidance Factor).3. Behavioral Incentive LoopTo solve the "human factor" challenge in hospitals, the platform includes a Staff Profit Share model. A percentage of the recovered value is logged to the specific staff member's ID, encouraging accurate segregation through micro-incentives.Technical ArchitectureFrontend: HTML5, CSS3, JavaScript (Vanilla)AI Layer: TensorFlow.js / Teachable Machine (Computer Vision)Database: Firebase Realtime DatabaseLogic: Dynamic ROI calculation and automated ledger logging.Financial Logic & FormulasTotal Revenue ($R_t$): $\sum (Quantity \times Market Value)$Disposal Savings ($S_d$): $R_t \times 0.8$ (Based on local incineration penalty rates)Net ROI: $\frac{(R_t + S_d) \times \text{Efficiency Variable}}{R_t}$Future Roadmap (Phase 2)Contamination Detection: Enhancing AI to detect biological fluids on recyclables.Dynamic Market API: Replacing hardcoded rates with live scrap market data from the Dhaka region.Offline-First Sync: Using localStorage to allow scanning in hospital zones with poor connectivity.
