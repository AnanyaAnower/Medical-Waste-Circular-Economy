**MedCycle AI: Intelligent Ledger for Medical Waste Resource Recovery**

**The Problem:**
As highlighted in the New Clean Dhaka Master Plan 2018-2032, Dhaka is facing a critical landfill shortage. Medical waste management costs are rising, and hospitals currently pay high fees for hazardous waste disposal. There is a significant gap in "intermediate treatment" and source segregation.

**The Solution:**
MedCycle AI is a circular economy framework that transforms hospital waste from a financial liability into a resource. By using Computer Vision (AI) and a Real-Time Firebase Ledger, the system identifies recyclable medical polymers at the point of disposal and calculates their economic value.

**Key Features:**
1. AI-Driven Material Valuation: The system uses a two-gatekeeper model to identify recyclables (Saline bags, IV tubes, Plastics). Unlike static systems, this project introduces a Quality Grading Scale (A-E)—inspired by technical assessment methodologies—to price waste based on its purity and condition.
   
2. Automated ROI Engine: The dashboard calculates a real-time Return on Investment based on:
   a) Direct Revenue: Market value of recovered high-grade medical polymers.
   b) Avoided Costs: Savings generated by diverting waste from expensive hazardous incineration streams
   (using a 0.8 Cost-Avoidance Factor).
   
3. Behavioral Incentive Loop: To solve the "human factor" challenge in hospitals, the platform includes a Staff Profit Share model. A percentage of the recovered value is logged to the specific staff member's ID, encouraging accurate segregation through micro-incentives.

**Technical Architecture:**
a) Frontend: HTML5, CSS3, JavaScript (Vanilla)
b) AI Layer: TensorFlow.js / Teachable Machine (Computer Vision)
c) Database: Firebase Realtime Database
d) Logic: Dynamic ROI calculation and automated ledger logging.

**Financial Logic & Formulas:** 
a) Total Resource Revenue, R_t = \sum (Quantity \times Market Value)
b) Disposal Savings, S_d = R_t \times 0.8 (Based on local incineration penalty rates)
c) Net ROI (%): \text{Net ROI} = \frac{(R_t + S_d) \times \text{Efficiency Variable}}{R_t} \times 100

**Future Roadmap (Phase 2):**
a) Contamination Detection: Enhancing AI to detect biological fluids on recyclables.
b) Dynamic Market API: Replacing hardcoded rates with live scrap market data from the Dhaka region.
b) Offline-First Sync: Using localStorage to allow scanning in hospital zones with poor connectivity.
